ResilientPlotterClass
=====================
A Python package to create resilient plots for geospatial data. This package is a wrapper around the plotting functionalities of `xarray <http://xarray.pydata.org/en/stable/>`_, `xugrid <https://deltares.github.io/xugrid/>`_, `geopandas <https://geopandas.org/>`_, `matplotlib <https://matplotlib.org/stable/index.html>`_, `contextily <https://contextily.readthedocs.io/en/latest/>`_, `cartopy <https://scitools.org.uk/cartopy/docs/latest/>`_ and `folium <https://python-visualization.github.io/folium/latest/>`_.

GitHub: https://github.com/Deltares-research/ResilientPlotterClass
