Documentation
-------------

The ResilientPlotterClass is a python package to create resilient plots for geospatial data. This package is a wrapper around the plotting functionalities of `xarray`_,
`xugrid`_, `geopandas`_, `matplotlib`, `contextily`_, `cartopy`_ and `folium`_.

.. _xarray: http://xarray.pydata.org/en/stable/
.. _xugrid: https://deltares.github.io/xugrid/
.. _geopandas: https://geopandas.org/
.. _matplotlib: https://matplotlib.org/stable/index.html
.. _contextily: https://contextily.readthedocs.io/en/latest/
.. _cartopy: https://scitools.org.uk/cartopy/docs/latest/
.. _folium: https://python-visualization.github.io/folium/latest/

The documentation of the ResilientPlotterClass consists of two parts :doc:`Guides <_pages/guides>` with detailed instructions on how to install and use the ResilientPlotterClass,
and :doc:`API reference <_pages/api_reference>` detailing every module, class and function implemented in the ResilientPlotterClass.
